## Projects

### 01 — CV Chatbot
**Stack:** TypeScript · Express · Gemini 2.0 Flash · React · Vite  
**What it does:** Parses a CV into structured data using Gemini's JSON schema output, then hosts a stateful chat interface for Q&A against the CV. Supports Google Drive OAuth for document ingestion.  
**Key patterns:** Structured output with `responseSchema`, session-cached chat, conversation history management  
**Status:** ✅ Complete
